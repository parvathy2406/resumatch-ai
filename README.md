# ResuMatch AI 🚀
An AI-powered resume optimization tool that analyzes resumes against job descriptions using the Groq LPU™ Inference Engine.

## Features
* Resume Analysis: Instant feedback on resume-to-job description alignment.
* ATS Optimization: Identifies missing keywords and suggests formatting improvements.
* High-Speed Inference: Powered by Groq's Llama 3 models for near-instant results.

## Tech Stack
* Frontend: React.js (Lucide-React, Axios)
* Backend: Python (Flask, Flask-CORS)
* AI Model: Groq API (Llama 3)
